# Local-First Conf 2026 transcripts

Transcripts and summaries of recorded talks from Local-First Conf 2026 in Berlin.

Each talk has three Markdown artifacts with the same filename stem:

- [transcripts/raw](transcripts/raw) contains timestamped output from whisper.cpp.
- [transcripts/cleaned](transcripts/cleaned) contains conservatively edited transcripts.
- [transcripts/summaries](transcripts/summaries) contains an overview and chronological notes.

The source recordings are kept locally in `recordings/` and are not tracked by Git.
